Descripción detallada

Declaración de los métodos generales de la API del CRFS.

Fecha
27 May 2020 Se incluyen los métodos para preparar el sistema de archivos y realizar operaciones de bajo nivel, dejando operaciones más sofisticadas que se aplican a nivel de archivo a otras parte de la API.

◆ cr_bitmap()

void cr_bitmap ( unsigned  partition_uid,
bool  hex 
)

Funcion para imprimir el bitmap.

Se imprime a stderr el estado actual del disco indicado, ya sea en hexadecimal o binario. Se puede indicar el partition 0 para ver el bitmap completo.

Parámetros
[in]partition_uidIndice de la partición a revisar, partiendo desde 1. Si es 0, se imprimen todos.
[in]hexSi se imprime el bitmap en formato hexadecimal o, por el contrario, binario.

◆ cr_mount()

int cr_mount ( char *  diskname)

Funcion para montar el disco.

Instancia la variable global disk_path con la ruta donde se encuentra el archivo .bin correspondiente al disco.

Si retorna 0, la operación se completo correctamente. En caso contrario, ocurrio un error.

Parámetros
[in]disknameNombre del archivo disco virtual
Devuelve
Estado de la operación
