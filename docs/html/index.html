Introducción

Esta documentación corresponde a un documento preparado para el ramo IIC2333, Sistemas Operativos y Redes, en el contexto del primer proyecto del primer semestre 2020. El equipo de desarrollo esta compuesto de los estudiantes Julio Andrade, Vicente Juárez, Diego Silva y Matías Silva.

Archivos Importantes

Los siguientes archivos son relevantes para el proyecto.

Archivos de API

La API se separa en src/cr_API/cr_API.h y src/cr_API/files_api.h

Archivos del Ejecutable

El código del ejecutable se encuentra en src/crfs/main.c

Archivos Internos del Sistema

Los archivos internos del sistema se separan en src/fileSystem/directory.h, src/fileSystem/index.h, src/fileSystem/bitmap.h, src/fileSystem/loaded_disk.h y src/fileSystem/partition.h.
