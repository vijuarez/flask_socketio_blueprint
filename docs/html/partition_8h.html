Descripción detallada

Declaración de los métodos para particiones del CRFS.

Fecha
27 May 2020 Se incluyen los métodos para manejar bloques índice en el CRFS, las estructuras de datos encargadas de representar a los archivos de forma individual en el sistema. Por esta razón, estas funciones se pueden ver como funciones para manejar archivos individuales. Además, son de bajo nivel, por lo que solo deberían ser usadas por un usuario por medio una API.
