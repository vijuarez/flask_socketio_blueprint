Descripción detallada

Declaración de los métodos para representación de disco en el CRFS.

Fecha
28 May 2020 Se incluyen los métodos para manejar un disco virtual entero en el CRFS, incluyendo representación e inicialización de este. Este conjunto de funciones son esenciales para preparar al CRFS para otras operaciones más sofisticadas.

◆ free_disk()

void free_disk ( Disk *  disk)

Función para liberar la memoria usada por un disco.

Esta función se encarga de liberar la memoria utilizada por una representación de un disco, descargando el disco por completo. Notese que esta función no actualiza el disco, solo se encarga de liberarlo.

Parámetros
[in,out]diskEl puntero del disco a liberar.

◆ init_disk()

Disk* init_disk ( char *  disk_path)

Función para cargar un disco en memoria.

Esta función recibe la dirección de un disco virtual, la cual es cargada en memoria como el sujeto para el CRFS. La totalidad del disco es cargado en memoria, y este se actualiza en el disco duro una vez el sistema termine su operación.

Parámetros
[in]disk_pathLa ruta del disco virtual.
Devuelve
Un puntero a la representación del disco.

int update_disk ( Disk *  disk,
char *  disk_path 
)

Función para actualizar un disco virtual.

Esta función se encarga de copiar el estado actual del CRFS a un disco virtual. Esta operación puede ser demorosa, dependiendo del tamaño del disco y la capacidad del computador.

Si retorna 0, la operación se realizo correctamente. Si retorna 1, ocurrió un error de escritura.

Parámetros
[in]diskEl puntero del disco a actualizar.
[in]disk_pathLa ruta del disco virtual.
Devuelve
El resultado de la operación.
