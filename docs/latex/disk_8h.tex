Declaración de los métodos para representación de disco en el C\+R\+FS. 

\begin{DoxyDate}{Fecha}
28 May 2020 Se incluyen los métodos para manejar un disco virtual entero en el C\+R\+FS, incluyendo representación e inicialización de este. Este conjunto de funciones son esenciales para preparar al C\+R\+FS para otras operaciones más sofisticadas. 
\end{DoxyDate}

Función para cargar un disco en memoria. 

Esta función recibe la dirección de un disco virtual, la cual es cargada en memoria como el sujeto para el C\+R\+FS. La totalidad del disco es cargado en memoria, y este se actualiza en el disco duro una vez el sistema termine su operación.

Función para actualizar un disco virtual. 

Esta función se encarga de copiar el estado actual del C\+R\+FS a un disco virtual. Esta operación puede ser demorosa, dependiendo del tamaño del disco y la capacidad del computador.

Si retorna 0, la operación se realizo correctamente. Si retorna 1, ocurrió un error de escritura.
