\doxysubsection{Descripción detallada}
Declaración de los métodos para representación de disco en el C\+R\+FS. 

\begin{DoxyDate}{Fecha}
28 May 2020 Se incluyen los métodos y representación para manejar el bloque bitmap de una partición del C\+R\+FS. Debido a la información que este tipo de bloque contiene, se debe mantener constantemente actualizado para representar correctamente el estado del disco. Por esto, se espera que estos métodos se ocupen tanto de forma interna por el sistema (para mantener la información actualizada) como externa (expuesto por la A\+PI para conocer el espacio libre). 
\end{DoxyDate}

Función para instanciar un bloque de bitmap. 

Esta función recibe un array de integers, los cuales son interpretados como un un bloque de bitmap. Cada bit de esta esta array indica si cierto bloque esta disponible para escribir (0) o esta ocupado (1).

Si el array no es válido, se devuelve un puntero N\+U\+LL.
